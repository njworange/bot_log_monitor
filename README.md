# 봇 로그 모니터

GDS Tool 로그 파일을 모니터링하여 자동으로 구글 드라이브 파일/폴더를 복사합니다.

## 주요 기능

- 📂 **실시간 로그 모니터링**: GDS Tool 로그를 실시간으로 감시
- 🎬 **영화 자동 복사**: TMDB API로 장르, 국가, 포스터 자동 수집
- 📺 **국내 VOD 자동 복사**: TMDB TV API로 프로그램 포스터 자동 수집
- 🌍 **외국 VOD 자동 복사**: 국내 VOD와 완전 분리된 별도 시스템
- 📁 **폴더 복사 기능**: 에피소드 하나 감지 시 프로그램 전체 폴더 복사 (v1.0.7+)
  - 이전 항목도 지원: 로그 자동 검색으로 경로 복원 (v1.0.9+)
- 🚫 **경로 무시 규칙**: 특정 경로의 파일을 자동으로 제외 (국내/외국 각각 설정)
- 🔄 **과거 로그 스캔**: 플러그인 시작 시 과거 로그 처리
- 🎯 **스마트 필터링**: 블랙리스트/화이트리스트 기반 자동 필터링 (국내/외국 각각 설정)
- 🚀 **GDS Tool 연동**: 자동 복사 요청 및 상태 추적
- 🔔 **Discord 웹훅 알림**: 복사 요청 및 에러 발생 시 실시간 알림

## 설정 방법

### 1. 기본 설정
- **로그 파일 경로**: GDS Tool 로그 파일 경로 (기본값: `/data/log/gds_tool.log`)
- **과거 로그 스캔 줄 수**: 플러그인 시작 시 스캔할 과거 로그 줄 수
- **자동 시작**: FlaskFarm 시작 시 로그 모니터링 자동 시작

### 2. TMDB 설정 (선택사항)
영화의 장르, 국가, 포스터 정보를 자동으로 가져오려면 TMDB API 키가 필요합니다.

**TMDB API 키 발급 방법:**
1. [TMDB 회원가입](https://www.themoviedb.org/signup) (무료)
2. [설정 > API](https://www.themoviedb.org/settings/api) 메뉴에서 API 키 신청
3. "API Key (v3 auth)" 복사하여 설정에 입력

**주의:** API 키 없이도 플러그인은 작동하지만, 영화의 장르/국가 정보가 비어있게 됩니다.

### 3. Discord 알림 설정 (선택사항)
복사 요청 및 에러 발생 시 Discord로 실시간 알림을 받을 수 있습니다.

**Discord 웹훅 URL 생성 방법:**
1. Discord에서 알림을 받을 채널 선택
2. 채널 설정 > 연동 > 웹훅 보기
3. "새 웹훅" 버튼 클릭
4. 웹훅 이름 설정 후 "웹훅 URL 복사"
5. 모니터 > 설정 > Discord 알림 탭에 붙여넣기

**알림 종류:**
- 🎬 국내 VOD 복사 요청 (초록색)
- 🌍 외국 VOD 복사 요청 (파란색)
- 🎥 영화 복사 요청 (주황색)
- ⚠️ 에러 발생 (빨간색)

### 4. 국내 VOD 설정
- **리모트 경로**: 복사 대상 경로 (예: `GoogleDrive:/process/download/download_tv`)
- **경로 무시 규칙**: 특정 경로의 파일을 처리하지 않습니다 (줄바꿈으로 구분)
  - 예: `/ROOT/GDRIVE/VIDEO/방송중/애니메이션/`
  - 예: `/ROOT/GDRIVE/VIDEO/방송중/교양/`
- **다운로드 모드**: none / blacklist / whitelist
- **필터링 설정**: 장르별, 프로그램별 필터링

### 5. 외국 VOD 설정
- **리모트 경로**: 복사 대상 경로 (예: `GoogleDrive:/process/download/download_tv_foreign`)
- **경로 무시 규칙**: 특정 경로의 파일을 처리하지 않습니다 (줄바꿈으로 구분)
  - 예: `/ROOT/GDRIVE/VIDEO/방송중/외국/미드/`
  - 예: `/ROOT/GDRIVE/VIDEO/방송중/외국/애니메이션/`
- **다운로드 모드**: none / blacklist / whitelist
- **필터링 설정**: 장르별, 프로그램별 필터링
- **자동 분류**: 다음 경로의 파일은 자동으로 외국 VOD로 분류됨
  - `/방송중/외국/` (예: 중드, 미드, 일드 등)
  - `/외국TV` (통합 외국 TV 폴더)

### 6. 영화 (S-MOVIE) 설정
- **리모트 경로**: 복사 대상 경로 (예: `GoogleDrive:/process/download/download_movie`)
- **다운로드 모드**: none / blacklist / whitelist
- **필터링 설정**: 장르별, 국가별, 연도별 필터링

## 사용법

1. **모니터 탭**: 로그 모니터링 시작/중지
2. **국내 VOD 탭**: 국내 TV 프로그램 필터링 설정 및 수신 목록 확인
3. **외국 VOD 탭**: 외국 TV 프로그램 필터링 설정 및 수신 목록 확인
4. **S-MOVIE 탭**: 영화 필터링 설정 및 수신 목록 확인

### 폴더 복사 사용법 (v1.0.7+)

VOD 목록에서 개별 에피소드가 감지되었을 때, "폴더 복사" 버튼으로 프로그램 전체 폴더를 한 번에 복사할 수 있습니다.

**사용 예시:**
```
"런닝맨.E789.260208.mp4" 파일이 감지됨
→ 목록에서 "폴더 복사" 버튼 클릭
→ "런닝맨 (2010) [SBS]" 폴더 전체 복사
→ 1화부터 마지막화까지 모든 에피소드가 한 번에 복사됨
```

**이전 항목도 지원 (v1.0.9+):**
- v1.0.7 이전에 수신된 항목도 폴더 복사 가능
- GDS Tool 로그를 자동으로 검색하여 경로 복원
- 로그에서 찾을 수 없는 경우 안내 메시지 표시
